# Second Brain Document Viewer

This folder contains all the living documents that Clara creates while working with Aniket. Each file is a markdown (`.md`) that captures:

- **Daily journal entries** – high‑level summaries of our conversations, decisions, and tasks.
- **Concept deep‑dives** – longer explorations of important topics we discuss (e.g., trading strategies, automation tricks, AI model choices).
- **Project notes** – specifications, sketches, and any artefacts that need a dedicated page.

## How it works
1. After each chat, Clara will automatically create or update a markdown file in this folder.
2. The file name follows the pattern `YYYY‑MM‑DD.md` for daily journals and `concept‑<topic>.md` for deeper topics.
3. All files are version‑controlled in the GitHub repo `https://github.com/aniketdubey1204-code/second_brain.git` (branch **main**).
4. You can view them directly via the Vercel site (`https://secondbrainn-ivory.vercel.app/`) – the viewer pulls the latest markdown from this folder.
